# TripGenie AI ✈️

TripGenie AI is an AI-powered travel planning application that generates personalized travel itineraries using Google Gemini AI. Users enter their destination, travel dates, budget, and preferences, and the application creates a customized travel plan within seconds.

## Features

- 🤖 AI-powered itinerary generation
- 🗺️ Interactive map preview
- 📅 Day-by-day travel itinerary
- 💰 Budget estimation
- 🛡️ Safety insights
- ♿ Accessibility information
- 🌱 Carbon footprint estimation
- 🔊 AI Voice Assistant
- 📄 PDF export
- 🔄 Regenerate itinerary
- 📱 Responsive design

## Tech Stack

- React
- TypeScript
- Tailwind CSS
- Vite
- Google Gemini AI

## How It Works

1. Enter your destination and travel preferences.
2. AI generates a personalized itinerary.
3. Review travel insights, budget, and recommendations.
4. Download the itinerary as a PDF or listen using the AI Voice Assistant.
